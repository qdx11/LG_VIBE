# Project Documentation Guide

이 프로젝트는 광물 선별 공정 데이터를 활용해 최종 품질을 예측하는 머신러닝 프로젝트다.

## Folder Rules

- `docs/log`: 사용자 질문과 답변, 주요 작업 내용을 간단한 worklog로 기록합니다.
- `docs/plan`: 프로젝트 방향, 단계별 계획, 모델링 전략을 기록합니다.
- `docs/data`: 데이터 출처, 구조, 라벨 정의, EDA에서 발견한 내용을 기록합니다.
- `docs/stages`: 각 stage의 실행 결과와 다음 결정사항을 기록합니다.
- `code`: 앞으로 작성하는 분석, 전처리, 모델링, 대시보드 코드를 저장합니다.
- `data`: Kaggle 등에서 받은 원본/가공 데이터 저장 위치입니다.
- `output`: 시각화 이미지와 stage별 실행 산출물을 저장합니다.
